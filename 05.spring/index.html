Git学习笔记

  1. 安装Git 在开始学习Git之前，需要在电脑上安装Git。可以从Git官网下载对应操作系统的安装包进行安装。

  2. 创建仓库 在本地创建一个Git仓库，使用git init命令。这样就可以开始管理代码了。

  3. 添加文件 将需要管理的文件添加到仓库中，使用git add命令。可以使用git add .命令来添加所有文件。

  4. 提交文件 将文件提交到仓库中，使用git commit命令。可以使用git commit -m “提交说明"命令来提交文件并添加提交说明。

  5. 查看状态 可以使用git status命令来查看当前仓库的状态，包括新增、修改或者删除的文件。

  6. 版本回退 使用git log命令可以查看提交历史，使用git reset命令可以回退到之前的版本。可以使用git reset –hard HEAD^回退到上一个版本，也可以使用git reset –hard 版本号回退到指定版本。

  7. 分支管理 使用git branch命令可以查看分支，使用git checkout命令可以切换分支，使用git merge命令可以合并分支。

    7.1 创建分支 使用git branch 分支名命令可以创建分支。

    7.2 切换分支 使用git checkout 分支名命令可以切换分支。

    7.3 合并分支 使用git merge 分支名命令可以将指定分支合并到当前分支。

    7.4 删除分支 使用git branch -d 分支名命令可以删除分支。

  8. 远程仓库 使用git remote命令可以查看远程仓库，使用git push命令可以将本地仓库推送到远程仓库，使用git pull命令可以将远程仓库更新到本地。

8.1 添加远程仓库 使用git remote add 仓库名 仓库地址命令可以将远程仓库添加到本地仓库中。

8.2 推送到远程仓库 使用git push 仓库名 分支名命令可以将本地仓库的指定分支推送到远程仓库。

8.3 从远程仓库拉取 使用git pull 仓库名 分支名命令可以将远程仓库的指定分支拉取到本地仓库。

8.4 查看远程仓库 使用git remote -v命令可以查看本地仓库关联的远程仓库信息。

  1. 其他命令 还有一些其他的Git命令，例如git clone、git diff、git tag等。需要根据具体需求进行学习和使用。

    9.1 克隆仓库 使用git clone 仓库地址命令可以将远程仓库克隆到本地。

    9.2 查看文件差异 使用git diff 文件名命令可以查看文件差异。

    9.3 打标签 使用git tag 标签名命令可以给当前版本打标签。

    以上是我学习Git的笔记，希望对你有所帮助。Git是一个强大的工具，需要结合实际项目进行练习和深入学习。
